start: start-all.cmd
kiểm tra các file có trong hdfs: hdfs dfs -ls /
copy một file trên máy lên hdfs: hdfs dfs -copyFromLocal /path/to/local/file(đường dẫn của file local) /path/to/hdfs/folder (đường dẫn của file copy vào hdfs)
    Ví dụ hdfs dfs -copyFromLocal "E:\Downloadclone\vnedict.txt" /vnedict
copy một file trên hdfs về máy local:  hdfs dfs -copyToLocal /path/to/hdfs/folder /path/to/local/file(đường dẫn của file local)
đọc 1 file trên hdfs: hdfs dfs -cat /vnedict
chạy mỗi datanode: ./hadoop-daemon.sh start datanode (trước khi chạy phải cd cmd vào thư mục bin của hdfs)
kiểm tra namenode kết nối đến datanode chưa
stop hdfs: stop-all.cmd
xem địa chỉ ip: ipconfig (xong là chọn IPV4)
